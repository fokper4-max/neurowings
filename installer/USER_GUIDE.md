# NeuroWings - Руководство по установке

## Для пользователей

Это руководство для тех, кто хочет установить NeuroWings на свой компьютер.

## Системные требования

### Минимальные требования:
- **ОС:** Windows 8.1 или новее (10, 11, 12)
- **Архитектура:** 64-bit (x64)
- **RAM:** 8 GB
- **Место на диске:** 5 GB свободного места
- **Права:** Администратор (для установки)

### Рекомендуемые требования:
- **ОС:** Windows 10/11
- **RAM:** 16 GB или больше
- **GPU:** NVIDIA GPU с CUDA поддержкой (опционально, для ускорения)
- **Место на диске:** 10 GB

## Установка

### Шаг 1: Загрузка

Скачайте файл инсталлятора:
```
NeuroWings-2.0-Setup.exe
```

### Шаг 2: Запуск инсталлятора

1. Дважды кликните на `NeuroWings-2.0-Setup.exe`
2. Разрешите приложению вносить изменения (UAC)
3. Следуйте инструкциям на экране

### Шаг 3: Выбор компонентов

Инсталлятор предложит установить:

- ✅ **Основная программа** (обязательно)
- ⬜ **Python Runtime** (если у вас нет Python)
- ⬜ **Visual C++ Redistributable** (требуется для PyTorch)

**Рекомендация:** Оставьте все компоненты выбранными, инсталлятор автоматически определит, что нужно устанавливать.

### Шаг 4: Выбор папки установки

По умолчанию:
```
C:\Program Files\NeuroWings
```

Вы можете изменить папку, если нужно.

### Шаг 5: Установка

Инсталлятор автоматически:
1. Определит версию Windows
2. Установит Python (если нужно)
3. Установит Visual C++ (если нужно)
4. Установит NeuroWings
5. Создаст ярлыки

**Примечание:** Процесс может занять 5-15 минут в зависимости от того, что нужно установить.

### Шаг 6: Завершение

После установки вы найдете:
- Ярлык на рабочем столе: `NeuroWings`
- В меню Пуск: `NeuroWings → NeuroWings`

## Что устанавливается автоматически

### Windows 8.1
- Python 3.8.18 (64-bit)
- Visual C++ 2015 Redistributable
- NeuroWings 2.0

### Windows 10
- Python 3.10.13 (64-bit)
- Visual C++ 2019 Redistributable
- NeuroWings 2.0

### Windows 11 / 12
- Python 3.11.8 (64-bit)
- Visual C++ 2022 Redistributable
- NeuroWings 2.0

## Первый запуск

1. Дважды кликните на ярлык `NeuroWings`
2. При первом запуске может потребоваться время на инициализацию моделей
3. Программа готова к работе!

## Удаление

### Через Панель управления:
1. Панель управления → Программы → Удалить программу
2. Найдите `NeuroWings`
3. Нажмите "Удалить"

### Через Settings (Windows 10/11):
1. Настройки → Приложения → Установленные приложения
2. Найдите `NeuroWings`
3. Нажмите три точки → Удалить

**Примечание:** Python и Visual C++ не будут удалены, так как могут использоваться другими программами.

## Устранение проблем

### Программа не запускается

**Проверьте:**
1. Установлены ли все компоненты?
2. Не блокирует ли антивирус?
3. Достаточно ли прав (попробуйте "Запуск от имени администратора")?

**Решение:**
- Переустановите программу
- Добавьте в исключения антивируса
- Установите последние обновления Windows

### Ошибка "DLL not found"

**Причина:** Не установлен Visual C++ Redistributable

**Решение:**
1. Переустановите NeuroWings
2. Убедитесь, что выбран компонент "Visual C++ Redistributable"

### Ошибка "Python not found"

**Причина:** Python не установлен или не в PATH

**Решение:**
1. Переустановите NeuroWings
2. Убедитесь, что выбран компонент "Python Runtime"

### Медленная работа

**Причины:**
- Недостаточно RAM
- Нет GPU (используется CPU)
- Много фоновых процессов

**Решение:**
- Закройте ненужные программы
- Увеличьте RAM
- Используйте GPU NVIDIA с CUDA

### Программа вылетает при обработке

**Возможные причины:**
- Недостаточно памяти
- Поврежденное изображение
- Слишком большое разрешение

**Решение:**
- Уменьшите размер изображения
- Проверьте формат файла (поддерживаются: JPG, PNG, TIF)
- Увеличьте виртуальную память Windows

## Обновление

При выходе новой версии:
1. Удалите старую версию
2. Установите новую версию
3. Настройки сохраняются в `%APPDATA%\NeuroWings`
