# NeuroWings - Быстрый старт

## Для разработчиков - Сборка инсталлятора

### За 3 простых шага:

#### 1️⃣ Установите требования

```powershell
# Python 3.8+ (если еще нет)
# Скачать: https://www.python.org/downloads/

# NSIS 3.x (для создания инсталлятора)
# Скачать: https://nsis.sourceforge.io/Download
```

#### 2️⃣ Дважды кликните на файл

```
installer\build_installer.bat
```

#### 3️⃣ Готово!

Файл инсталлятора будет в:
```
dist\NeuroWings-2.0-Setup.exe
```

---

## Для пользователей - Установка программы

### За 2 простых шага:

#### 1️⃣ Запустите инсталлятор

Дважды кликните на:
```
NeuroWings-2.0-Setup.exe
```

#### 2️⃣ Следуйте инструкциям

Инсталлятор всё сделает автоматически:
- ✅ Определит версию Windows
- ✅ Установит Python (если нужно)
- ✅ Установит Visual C++ (если нужно)
- ✅ Установит NeuroWings
- ✅ Создаст ярлыки
