В папке `dist/` находятся готовые к использованию файлы:

## 📦 КАК СОЗДАТЬ ПОЛНЫЙ ИНСТАЛЛЯТОР (опционально)

Для создания профессионального инсталлятора с:
- Автоматической установкой ярлыков
- Деинсталлятором
- Регистрацией в "Программы и компоненты"

### Шаг 1: Установите NSIS

**Вариант A: Скачать вручную (рекомендуется)**
1. Перейдите: https://nsis.sourceforge.io/Download
2. Скачайте NSIS 3.10 (или новее)
3. Установите с правами администратора

**Вариант B: Через PowerShell с правами администратора**
```powershell
cd "c:\NeuroWings 2.0\installer"
.\download_and_install_nsis.ps1
```

### Шаг 2: Создайте инсталлятор

После установки NSIS просто запустите:
```batch
installer\build_simple_installer.bat
```

**Результат:**
```
dist\NeuroWings-2.0-Setup.exe (~555 MB)
```

## ❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

### Q: Нужно ли устанавливать Python пользователям?
**A:** Нет! EXE файл содержит всё необходимое.

### Q: На каких версиях Windows работает?
**A:** Windows 8.1, 10, 11, 12+ (только 64-bit)

### Q: Как обновить программу?
**A:** Пересоберите EXE командой:
```batch
pyinstaller --clean --noconfirm installer\NeuroWings.spec
```

### Q: Антивирус блокирует EXE
**A:** Это нормально для PyInstaller. Решения:
1. Добавьте в исключения антивируса
2. Используйте code signing сертификат (для профессионального распространения)
3. Отправьте файл на проверку в антивирусные компании

### Q: Как уменьшить размер EXE?
**A:** EXE большой из-за PyTorch (~400 MB). Возможные варианты:
- Использовать CPU-only версию PyTorch (меньше размер)
- Исключить неиспользуемые модули
- Использовать UPX сжатие (уже включено)
